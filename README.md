# ISLR-Ch7-8-Non-linear-and-tree-models-Labs
Comprehensive R implementation of Chapter 7.8 (Non-linear Modeling) and 8.3 (Decision Trees) Labs from An Introduction to Statistical Learning (ISLR, James et al., 2021). Features implementation of Non-linear Regression (Polynomials, Splines, LOESS), Generalized Additive Models (GAMs), CART, Bagging, Random Forests, Boosting (GBM), and BART.
